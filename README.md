# 🕵️‍♂️ UNVAIL

> **허위·과장 광고를 벗기다**  
> OCR 인증과 AI 리뷰 분석을 통해  
> **‘내돈내산’ 기반의 신뢰 가능한 소비 플랫폼**을 만듭니다.

---

## 📌 프로젝트 개요

**UNVAIL**은 SNS 및 온라인 플랫폼에 만연한  
허위·과장 광고 문제를 해결하기 위해 기획된 **AI 기반 신뢰 검증 플랫폼**입니다.

사용자가 업로드한 **광고 이미지 및 영수증**을 OCR로 분석하고,  
AI가 리뷰 신뢰도를 평가하여 **제품·기업의 신뢰 등급(A~F)**을 산출합니다.

> “믿고 샀는데 또 속였다”  
> 이제는 **데이터로 검증하는 소비**를 지향합니다.

---

## 🎯 해결하고자 하는 문제

- SNS 중심의 허위·과장 광고 급증
- 광고 후기와 실제 사용자 리뷰의 구분 불가
- 건강·식품·다이어트 제품 구매로 인한 소비자 피해
- 소비자가 신뢰할 수 있는 **객관적 기준의 부재**

---

## 💡 핵심 기능

### 1️⃣ OCR 기반 ‘내돈내산’ 인증
- 영수증 / 결제 내역 이미지 업로드
- OCR을 통한 상품명·결제 정보 자동 추출
- 실제 구매 여부 검증으로 리뷰 신뢰성 확보

### 2️⃣ AI 리뷰 신뢰도 분석
- LLM 기반 리뷰 텍스트 분석
- 광고성 표현, 패턴 반복, 어뷰징 탐지
- 리뷰어 활동 이력과 결합한 신뢰 점수 산출

### 3️⃣ 신뢰 등급 시스템 (A ~ F)
- 리뷰 품질, 인증 여부, 신뢰도 가중치 반영
- 모든 제품에 **일관된 평가 기준** 적용
- 직관적인 시각화 제공 (매너온도 방식)

### 4️⃣ 투명한 커뮤니티
- 영수증 인증 리뷰만 분리 노출
- 실시간 인증 리뷰 수, 좋아요 수 제공
- 광고 리뷰와 명확히 구분되는 UX

### 5️⃣ ‘착한 기업’ 보상 생태계
- A등급 기업 제품만 노출
- 신뢰 리뷰 활동에 따른 포인트 지급
- 윤리적·친환경 기업 제품 구매로 선순환 구조 형성
